# Immune System

This is a simple immune system simulation, it includes cells, DNA, and a few other things.

## How to use

1. Make sure you have python 3.6 or higher installed.
2. Clone the repository.
3. Run the `cellSim.py` file.

## Features

* Cells
* DNA
* Immune System
* Randomness
